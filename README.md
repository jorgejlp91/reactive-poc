# Getting Started

## Purpose of this Project

   This project was created for study the concepts of reactive programming

### Pre-Requisites
   * Java 1.8
   * Docker Composer
   * Maven 3.5 or higher

### Run Applications

   * Just run the follow comands bellow
   `docker-compose -f docker-compose.yml up --build --force-recreate -d`<br/>
    `mvn spring-boot:run`
